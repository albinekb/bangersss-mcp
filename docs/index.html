Features

Everything a DJ needs

Built for real DJ workflows. Safe by default with dry-mode operation — nothing touches your files until you say so.

Dry Mode

All operations go through an in-memory overlay filesystem. Preview changes, discard, or selectively commit — like git for your music library.

Exportable Plans

Create portable operation plans. Export as JSON, review, share with other DJs, or resume later. Organize 10,000 tracks with confidence.

BPM Analysis

Detect BPM from audio files using ffmpeg and autocorrelation analysis. Batch process your entire collection.

ID3 Tag Management

Read and write metadata tags on MP3, FLAC, WAV, AIFF, M4A, OGG. Auto-suggest missing fields.

Rekordbox Integration

Read and write to Rekordbox's encrypted database. Search tracks, manage playlists, read cue points and hot cues.

Engine DJ Integration

Connect to Engine DJ databases on USB drives or SD cards. Search tracks, manage crates, view library stats.

Key & Harmonic Mixing

Camelot wheel, Open Key, and standard notation. Check key compatibility and find harmonic neighbors for perfect mixes.

Smart File Organization

Organize files into folders using tag-based templates. {genre}/{artist}/{title}, {camelot}/{bpm}/{artist}, or any pattern you want.

Playlist Management

Create, edit, and export M3U/M3U8 playlists. Add and remove tracks, list all managed playlists.

How it works

Natural language, real results

CrateDigger is an MCP server — it connects to AI assistants like Claude Code and Claude Desktop, giving them tools to manage your music.

Connect

Add CrateDigger to your Claude Code or Claude Desktop MCP config. One line of JSON.

Describe

Tell Claude what you want in plain English. "Organize my downloads by genre" or "Find all tracks between 124-128 BPM."

Preview

Every change goes to dry mode first. Review the proposed moves, renames, and tag edits before anything happens.

Commit

Approve the changes. Files get moved, tags get written, playlists get updated — all at once, or selectively.

Get started

Up and running in minutes

Clone, build, and connect to your AI assistant.

# Install
git clone https://github.com/albinekb/musicsorter.git cratedigger
cd cratedigger
npm install && npm run build

# Add to Claude Code MCP settings
# ~/.claude/settings.json
{
  "mcpServers": {
    "cratedigger": {
      "command": "node",
      "args": ["/path/to/cratedigger/dist/index.js"]
    }
  }
}
